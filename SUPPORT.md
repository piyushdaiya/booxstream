# Support

If you need help using BooxStream, please use one of the following options.

---

## Documentation
Check the README and project documentation first.

---

## GitHub Issues
Open an issue for:
- bugs
- feature requests
- installation problems

Please include:
- device model
- Android version
- host OS
- logs if available

---

## Discussions
For general questions, consider using GitHub Discussions (if enabled).

---

## Community Contributions
Community contributions and improvements are always welcome.
